# Extension Icons

Place the following icon files here:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

These icons should be the DialogBrain logo in the appropriate sizes.

For development/testing, you can use placeholder icons. Chrome will still load the extension but show a default icon.

## Recommended Design

- Use the DialogBrain "brain" logo
- Solid color background (DialogBrain blue: #2563eb)
- White icon on blue background
- PNG format with transparency if needed
